
# .NET

## .NET Runtime

The modern cross-platform .NET runtime (formerly .NET Core 5+):

  * `net-runtime-10-x86`: .NET Runtime 10 (x86)
  * `net-runtime-10-x64`: .NET Runtime 10 (x64)
  * `net-runtime-9-x86`: .NET Runtime 9 (x86)
  * `net-runtime-9-x64`: .NET Runtime 9 (x64)
  * `net-runtime-8-x86`: .NET Runtime 8 (x86)
  * `net-runtime-8-x64`: .NET Runtime 8 (x64)
  * `net-runtime-7-x86`: .NET Runtime 7 (x86)
  * `net-runtime-7-x64`: .NET Runtime 7 (x64)
  * `net-runtime-6-x86`: .NET Runtime 6 (x86)
  * `net-runtime-6-x64`: .NET Runtime 6 (x64)
  * `net-runtime-5-x86`: .NET Runtime 5 (x86)
  * `net-runtime-5-x64`: .NET Runtime 5 (x64)

## .NET Core

Legacy .NET Core runtime (for older cross-platform applications):

  * `net-core-2-x86`: .NET Core 2 (x86)
  * `net-core-2-x64`: .NET Core 2 (x64)

## .NET Framework

Windows-only .NET Framework (for legacy applications):

  * `net-4.8`: Microsoft .NET Framework 4.8
  * `net-4.7.2`: Microsoft .NET Framework 4.7.2
  * `net-4.7.1`: Microsoft .NET Framework 4.7.1
  * `net-4.6.2`: Microsoft .NET Framework 4.6.2
  * `net-4.6`: Microsoft .NET Framework 4.6
  * `net-4.5.2`: Microsoft .NET Framework 4.5.2

The page [.NET Framework Versions and Dependencies](https://learn.microsoft.com/en-us/dotnet/framework/migration-guide/versions-and-dependencies) covers the differences between various versions.
